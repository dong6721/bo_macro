# Blue_oath-macro-(jp server)

## Blue_oath,푸른맹세 해역 자동순환 Macro


### 기능
> 특정 해역 반복 순환
> > 일정 연료 도달시 정지   
> > 드랍함 일정 회수 드랍 시 정지   
> > custom 지역 설정 가능   

자세한 사항은 도움말

### How to using tool
* initial screen
![Alt text](/init.PNG)
1. execute "bo_macro.exe"
2. set game screen to init screen
3. set macro's stage
4. push "F5" button
+ do not use mouse, this is mouse click base program.
